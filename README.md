# react-app
世界天気予報、タスク管理、カレンダー機能があります。
ハンバーガーメニュー実装しました。(ライブラリ使用)
CSSの変更も一部あり。
brotherrouterの記述更新、世界天気検索時に空でフォーム送信でリロードメッセージでリロード誘導のため、リロードボタン導入。
Suspenseで不要なコンポーネントを遅延レンダリング導入。
ページごとに背景が動的に変更されるように更新とTodo管理画面ではダークモード実装。
未完了タスクをExcelにエクスポートボタンを導入。
スマートフォンでのUI改善。
